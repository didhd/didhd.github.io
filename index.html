프리플랍

상대

플레이어

팟: 0

기본 규칙

텍사스 홀덤은 각 플레이어가 2장의 개인 카드(홀 카드각 플레이어에게 비공개로 주어지는 2장의 카드)를 받고, 테이블에 공개되는 5장의 공통 카드(커뮤니티 카드모든 플레이어가 공유하는 5장의 공개 카드)를 이용해 가장 높은 족보를 만듭니다.

게임 진행 순서: 프리플랍 → 플랍 → 턴 → 리버 → 쇼다운

  • 체크추가 베팅 없이 턴을 넘김
  • 벳처음으로 베팅 시작
  • 콜상대 베팅에 맞추어 베팅
  • 레이즈상대 베팅보다 높게 베팅
  • 폴드패를 포기
